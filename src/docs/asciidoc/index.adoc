=== The Gradle build cache demo

== CI Integration

=== Snapshots and CI

* Can setup composite builds on CI
** allows discovering breakages early
* Enable build cache!

[source,groovy]
----
ext.isCiServer = System.getenv().containsKey("CI")

buildCache {
    local {
        enabled = !isCiServer
    }
    remote(HttpBuildCache) {
        url = 'https://example.com:8123/build-cache/'
        push = isCiServer
    }
}
----
